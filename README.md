# **AltimetrikGameFinderApp**

## HOW TO RUN THE APP

Start Json server / Json server auth:

```
npx json-server db.json -m ./node_modules/json-server-auth
```

Accepted users:

```
{
  "email": "test@mail.com",
  "password": "hola"
}
```

Open gameFinder folder in visual Studio Code, and run login.html in live server.
